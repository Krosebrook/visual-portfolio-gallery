# Visual Portfolio Gallery

A stunning, immersive visual portfolio designed to showcase creative work with elegance and precision. Built with React, Tailwind CSS, and Framer Motion, featuring a seamless full-screen gallery experience.

## Features

- **Full-Screen Gallery**: Immersive slide-based viewing experience
- **Grid View**: Overview of all works with category filtering
- **Admin Panel**: Secure dashboard to manage projects, testimonials, and inquiries
- **Testimonials**: Dedicated section for client reviews
- **Contact Form**: Integrated inquiry system stored in database
- **Responsive Design**: Flawless experience across mobile and desktop

## Tech Stack

- **Framework**: Vite + React + TypeScript
- **Styling**: Tailwind CSS + Shadcn UI
- **Animations**: Framer Motion
- **Backend**: Blink SDK (Auth, Database, Storage)
- **Icons**: Lucide React

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
